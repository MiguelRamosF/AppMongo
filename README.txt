BEFORE LAUCNHING THE PROJECT:

-NodeJS must be installed

-Import cities.json in your mongoDB with the following command:
mongoimport --db ZIPS --collection cities --file path/cities.json

-Mongod must be launched



TO RUN THE PROJET:

1)Open a command prompt in the folder project
2)Run the following commands :
npm install
node index.js
3)Open your browser and go to localhost:4000/index.html
